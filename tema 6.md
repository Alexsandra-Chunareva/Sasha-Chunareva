### Ответ:
```python
def tuple_sort(tpl):
    for elm in tpl:
        if not isinstance(elm, int):
            return tpl
    return tuple(sorted(tpl))

if __name__ == '__main__':
    print(tuple_sort((5, 5, 3, 1, 9)))
    print(tuple_sort((5, 5, 2.1, '1', 9)))
```

### Ответ:
```python
from collections import Counter
def count_repeating_numbers(sequence):

    numbers_counts = Counter(sequence)
    most_common_numbers = numbers_counts.most_common(3)
    for key, value in sorted(most_common_numbers, key = lambda x: x[0]):
        print('Число:', key , 'Встречается столько раз:', value)

sequence = '955687904333784999'
count_repeating_numbers(sequence)
```

Если элемента нет вообще – вернуть пустой кортеж.

Если элемент встречается только один раз, то вернуть кортеж, который начинается с него и идет до конца исходного.

Входные данные:
(1, 2, 3), 8)
(1, 8, 3, 4, 8, 8, 9, 2), 8)

### Ответ:
```python
def cortesh(incoming_list, number):
    if number not in incoming_list:
        return (0)
    if incoming_list.count(number) == 1:
        return incoming_list[incoming_list.index(number):]
    return incoming_list[incoming_list.index(number):incoming_list.index(number, incoming_list.index(number) + 1) + 1]


print(cortesh((1, 2, 3), 8))
print(cortesh((1, 8, 3, 4, 8, 8, 9, 2), 8))
print(cortesh((1, 2, 8, 5, 1, 2, 9), 8))
```

Если число number отсутствует в списке, функция возвращает 0. В противном случае, если число встречается в списке ровно один раз, функция возвращает весь список после этого числа. ### Словари
Словарь представляет собой неупорядоченную коллекцию пар ключ-значение. У каждого элемента словаря есть уникальный ключ, который позволяет получить соответствующее значение. Создавать словари можно несколькими способами:
- Используя литералы (например, {key1: value1, key2: value2}).
- С помощью функции dict().
- Компрессией словаря (перевод существующего объекта в словарь).
- Методом fromkeys().

У словарей есть множество методов, таких как:
- keys(), values(), items(), позволяющие получить соответствующие коллекции ключей, значений и пар ключ-значение.
- get(), pop(), popitem(), позволяющие взаимодействовать с содержимым словаря.
- update(), clear(), позволяющие изменять словарь.

### Кортежи
Кортежи — это неизменяемые упорядоченные коллекции объектов. Они создаются с помощью круглых скобок (...), функции tuple(), а также иногда без скобок. К ним применимы следующие операции:
- Доступ к элементам.
- Срезы.
- Конкатенация.
- Проверка длины.
- Повторение.
- Проверка наличия элемента.

Кроме того, кортежи полезны в качестве параметров и результатов функций.

Таким образом, словари и кортежи играют важную роль в Python, предоставляя гибкие инструменты для работы с различными типами данных.
